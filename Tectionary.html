Variáveis de valor fracionário

Bool

Variáveis de valor lógico

If

Condição para ativação de determinada ação

Else

Condição oposta ao If

Readline

Comando para o sistema permitir o usuário digitar

Collider

Configuração para que o objeto colida com outros se encostar

Script

Nome dado a área em que se digita códigos de programação

Writeline

Comando para o sistema em si escrever algo

While

Comando para que uma determinada ação se repita se uma condição for cumprida

Rigidbody

Configuração para que um objeto sofra a ação da gravidade

For

Maneira alternativa de repetição de ações

Break

Comando para que o sistema pare de repetir a mesma ação

Render

Ação de carregamento do gráfico do cenário

Back end

uma parte da web que o usuário não consegue ver

Front end

A parte da web visível ao usuário

IoT

Significa “Internet of Things” ou “Internet das Coisas”, é um conceito que se refere a interligação inteligente das coisas, computadores, objetos, com o fim de facilitar a vida do usuário

Debug

É processo de “Debugar”, ler o que o software ou hardware está fazendo, em busca de falhas e informando cada passo das operações

Array

Significa “variedade” e é usado na programação como matriz, um gaveteiro de informações, guardando variáveis… a primeira “gaveta” de um Array, começa a ser numerada por 0

Default

Pode ser utilizado tanto para referir-se a um valor pré-definido, que o sistema computacional assume, como para uma ação pré-definida, tomada pelo sistema a menos que seja instruído de outra maneira.

Double

As variáveis do tipo double representam números de ponto flutuante precisam duas vezes a quantidade de memória das variáveis float fornecendo 15 dígitos, sendo o dobro da precisão de variáveis float.

Void

Ele nos permite fazer funções que não retornam nada e funções que não têm parâmetros

Case

É uma forma de reduzir a complexidade de vários if ou else encadeados. É muito utilizado, principalmente para uso em estruturas de menu

Switch

Serve para fazer testes condicionais, testando igualdades, onde podemos usar várias opções de comparações. Assim como o IF/ELSE, o comando switch também serve para fazer testes condicionais.

Vector3

É composto por 3 componentes "(x,y,z)" e basicamente serve para guardar 3 valores, um valor para cada um de seus componentes. Você pode usar um Vector3 em diversos casos, mas é comum ser usado para guardar ou setar posições dos objetos no espaço 3D
